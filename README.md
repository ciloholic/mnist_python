https://github.com/oreilly-japan/deep-learning-from-scratch

## version

- Python 3.6
- pipenv

## package install

```
pipenv install --dev
```

## flake8 or autopep8

```
pipenv run lint
pipenv run format
```

## run

```
pipenv run python mnist_divide.py
```
